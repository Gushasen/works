# CRYCHICは終わらせないといけないから

ルール１．「私」は自分の演技に気づかない。

ルール２．「いま、祥子ちゃんの一番近くにいるのは睦ちゃんだから。祥子ちゃんのこと、よろしくね」

ルール３．「睦ちゃん。いまの、祥子ちゃんには内緒だよ」

ルール４．「睦ちゃん」なんて最初からいないよ。

## 想像の大前提

「もし自由でなければ、自ら欲するところでなく、強いられて行うことだけが表面に現れる以上、彼らとしても真実な忠誠、変わらざる信義や愛、の真摯な証拠をいかにして示しえようか？」失楽園第三巻104行目-107行目

「運命、もしくは、確実不変のもの、としてわたしが予見したいかなるものの刺激や余波をいささかも受けることなく、彼らは自らの判断と自らの選択とにおいて罪を犯し、その主体となってゆくのだ」失楽園第三巻120行目-123行目

「自ら好んで奴隷とならない限り、彼らはあくまで自由な者として生きてゆかなければならない」失楽園第三巻124行目-125行目

若葉睦の物語を書いた人が失楽園の自由意志の思想を尊重しているものと信じて前提を置く。

睦の忠誠や信義や愛が真実であるために睦は自由であるように創られている。

睦自身の意志と理性による選択が必ずある。
